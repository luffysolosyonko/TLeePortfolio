# Thomas Lee - Portfolio

Personal portfolio website showcasing my software engineering projects, work experience, and technical skills.

🌐 **Live Site:** [https://luffysolosyonko.github.io/TLeePorfolio/](https://luffysolosyonko.github.io/TLeePorfolio/)

## Tech Stack

- **Vue 3** - Progressive JavaScript framework
- **Vite** - Lightning-fast build tool
- **GitHub Pages** - Hosting & deployment

## Features

-  NVIDIA-inspired dark theme with green accents
-  Fully responsive design
-  Inline YouTube video modals for project demos
-  Embedded resume viewer
-  Optimized performance with lazy loading

## Project Setup

Install dependencies:
```sh
npm install
```

### Development

Run local dev server with hot reload:
```sh
npm run dev
```

### Build for Production

```sh
npm run build
```

### Deploy to GitHub Pages

```sh
npm run deploy
```
